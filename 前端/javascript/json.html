  • json
    格式:
    a)键值对使用：{}，使用，分割
    b)数组使用：[]，使用，分割
    如：
    <script type="text/javascript">
        var people = {
                "head" : "yuande",
                "body" : [
                            {    "wear" : "short"} ,
                            {    "shoe" : "nike"}
                            ]
                };
    </    script> 转化：
    a) 将字符串转化为json对象
        var str = "{'head' : 'yuande','body' : [{'wear' : 'short'} ,{'shoe' : 'nike'}]}";
        var     data = eval( '(' + str + ')' );
    b) 使用插件 json2.js 将json转化为字符串
        var text = JSON.stringify( data );
